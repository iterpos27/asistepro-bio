# AsistePro SaaS Docs

Esta carpeta contiene documentación base para crear el proyecto **AsistePro**, un SaaS Multi-Tenant de asistencia laboral mediante QR + GPS.

## Archivos

- `ASISTEPRO_SAAS_V1.md`: Documento funcional y técnico del producto.
- `PROMPT_IA_ASISTIDA_ASISTEPRO.md`: Prompt maestro para usar en una IA asistida de desarrollo.
- `BASE_DE_DATOS_ASISTEPRO.sql`: Script SQL inicial sugerido.

## Uso recomendado

1. Abre `PROMPT_IA_ASISTIDA_ASISTEPRO.md`.
2. Copia el contenido completo.
3. Pégalo en Cursor, Windsurf, GitHub Copilot Agent, Claude Code u otra IA asistida.
4. Pide que empiece por la FASE 1.
5. Valida cada fase antes de continuar.
