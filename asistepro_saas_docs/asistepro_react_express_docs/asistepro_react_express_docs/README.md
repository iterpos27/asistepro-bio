# AsistePro - Documentación para IA Asistida

Este paquete contiene la reformulación del proyecto AsistePro usando:

- React + Vite
- Node.js + Express
- PostgreSQL
- Prisma ORM
- TypeScript

Archivos incluidos:

1. `ASISTEPRO_REACT_EXPRESS_V1.md`
   - Documento funcional y técnico del SaaS.

2. `PROMPT_IA_ASISTIDA_ASISTEPRO_REACT_EXPRESS.md`
   - Prompt maestro para usar en Cursor, Windsurf, Claude Code, Copilot Agent u otra IA de desarrollo.

3. `PRISMA_SCHEMA_ASISTEPRO.prisma`
   - Modelo inicial de base de datos para Prisma + PostgreSQL.

Recomendación:

Primero entrega el prompt maestro a la IA asistida y pídele que empiece únicamente con la FASE 1.
